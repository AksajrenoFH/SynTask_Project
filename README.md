## SynTask Profile

🇮🇩 <br/>
SynTask adalah platform pembelajaran coding yang dirancang untuk **membantu pengguna belajar secara interaktif dan menarik**, dengan menggabungkan teori dan praktik. 
Pengguna dapat mengikuti roadmap yang terstruktur untuk meningkatkan pengalaman, keterampilan, serta kemampuan problem solving secara bertahap.
SynTask ditujukan untuk **remaja dan pemula**, sebagai jembatan untuk mulai memahami dunia pemrograman.

🇬🇧 <br/>
SynTask is a learning platform designed **to help users learn programming in an interactive and engaging way** by combining theory and practice.
Users can follow a structured roadmap to gradually improve their skills, experience, and problem-solving abilities.
SynTask is built **for teenagers and beginners**, aiming to provide a bridge for users to start developing their programming skills.

## 🚀 Getting Started

Follow these steps to run the project locally:

1. Install dependencies

```
npm install
```
2. Start the development server

```
npx expo start
```

After running the app, you can open it using:
- Android Emulator
- iOS Simulator 
- Expo Go (for quick testing on your device)
